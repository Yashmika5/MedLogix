Data Structures and Algorithms Description
Array: Statically stores category names, supports indexing and iteration.

Linked List: Medicines for each category, enables dynamic insertions and deletions with pointer-based navigation.

Binary Tree: Medicine reminders scheduled by time, for O(log n) insertion/search and sorted print (using in-order traversal).

Queue: Holds reminders, processed in strict time order (first entered, first out).

Stack: Tracks each change for undo operations.
